    1+ year as Data Scientist

    During this period, I worked for two companies: Milestone Consulting and Newbacon.

    At Newbacon, I contributed in building a product-based recommendation system. I also contributed in building a model to predict stock and implementing the model in production. I built models for Churn prediction based on Machine Learning, performed some Market basket analysis (MBA). I performed cluster segmentation using the following techniques: RFM, RFV+, K-Means. The tools I used the most during this period were: Python, SQL, Pyspark.

    At Milestone Consulting, I participated in a project that had the objective of developing a model to forecast the price of energy in the Iberian Market. This internship added a lot to my professional career, because I worked on this project from end to end, so I could see all the stages of a project. I used time series techniques (ARIMA, SARIMA and ARIMAX) and neural networks to build the forecast models. Also, I developed a model to predict Market Splitting, a phenomenon in which is characterized when there is a price difference in electric energy between Spain and Portugal. This model was built using Logistic Regression. The tools I used during this internship were: R, Stata, IBM SPSS and Microsoft Excel.
